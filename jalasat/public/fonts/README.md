# فونت لوکال پروژه

طبق درخواست، این پروژه به هیچ CDN بیرونی (نه گوگل‌فونت، نه CDN آماده) برای فونت وصل نیست؛
فونت باید کاملا لوکال (self-hosted) از همین پوشه سرو شود.

## مرحله‌ی لازم قبل از اجرا

فایل‌های فونت وزیرمتن (Vazirmatn) - یکی از بهترین فونت‌های فارسی متن‌باز - را از صفحه‌ی رسمی
گیت‌هاب پروژه (‍github.com/rastikerdar/vazirmatn، بخش Releases) دانلود کرده و فایل‌های زیر را
داخل همین پوشه (public/fonts) قرار دهید:

```
Vazirmatn-Regular.woff2
Vazirmatn-Medium.woff2
Vazirmatn-Bold.woff2
```

فایل src/styles/theme.css از قبل با @font-face به همین نام‌ها لینک شده است، پس کافیست فایل‌ها
را کپی کنید؛ نیازی به تغییر کد نیست. تا قبل از اضافه‌کردن فایل‌ها، سیستم به‌صورت خودکار از فونت
سیستم‌عامل (Tahoma) استفاده می‌کند تا ظاهر خراب نشود.

اگر ترجیح می‌دهید از فونت اختصاصی سازمانی (مثلا فونت لوگوی ادب‌ویزا) استفاده کنید، فقط کافیست
همین سه فایل woff2 را با فونت دلخواه جایگزین کنید.
